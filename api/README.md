This folder contains the code for the API used to fetch the URLs of VODs that can't be embedded via an iframe and have dynamic URLs that can't be fetched from the frontend.

It was made with [Deno](https://deno.com) and runs on [Alwaysdata](https://alwaysdata.com) at [vod-api.escfent.com](https://vod-api.escfent.com).
